## Запуск приложения

### Docker
- создать файл .env
- указать переменные
    -- PORT - порт для запуска приложения
    -- DB_PASSWORD - пароль для базы данных !обязательно!
    -- DB_USER - имя пользоваеля базы данных (по умолчанию postgres)
    -- DB_NAME - название базы данных (по умолчанию postgres)
